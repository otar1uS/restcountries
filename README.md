### Cloning the repository

```shell
gh repo clone otar1uS/restcountries
```

### Install packages

```shell
npm i
```

```shell
npm run dev
```

## Available commands

Running commands with npm `npm run [command]`

| command | description                              |
| :------ | :--------------------------------------- |
| `dev`   | Starts a development instance of the app |
